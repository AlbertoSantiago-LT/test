Subir cambios Ejemplo
    - git add .
    - git commit -m "Mensaje de commit"
    - git push origin main

Crear una rama
    - Te crea la rama y te cambia esa rama
        git checkout -b Nueva-Rama-Nombre

Añadir cambios
    - Añadir todos los cambios
        git add . 
    - Añadir solo los cambios que quieres del fichero que selecciones
        git add carpeta
Añadir Commit 
    - Añadir mensaje del commit
        git commit -m "Mensaje"
Obtener cambios de repositorio
    - Obtener cambios
        git pul origin rama-que-quieres
Subir cambios
    - Subir todos los cambios
        git push origin rama-que-quieres-subir
